# Cyber Invasion Army Portfolio

Static portfolio website for Cyber Invasion Army (CIA), a CTF team.

## Structure

- `index.html` redirects GitHub Pages visitors to the main portfolio page.
- `cia-portfolio.html` contains the site markup, styles, and scripts.
- `gallary/` contains published gallery images.
- `AGENTS.md` contains contributor guidelines.

## Local Preview

Run a local static server from the repository root:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## GitHub Pages

This site can be hosted from the repository root on the `main` branch using GitHub Pages.
